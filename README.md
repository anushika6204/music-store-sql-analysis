# 🎵 Music Store Data Analysis

## 📌 Project Overview
Analyzed a music store's relational database to identify
top-performing artists, popular genres, and business growth
opportunities using advanced SQL queries.

## 🛠️ Tools Used
- PostgreSQL — Database Querying
- MySQL — Data Analysis
- VS Code / pgAdmin

## 📊 Key Results
- ✅ Analyzed 11 tables with 3,500+ records
- ✅ Wrote 20+ complex SQL queries
- ✅ Used CTEs, Window Functions & Subqueries
- ✅ Identified top artists, genres & revenue trends
- ✅ Helped store understand business growth patterns


## 📁 Files in this Repository
| File | Description |

| `Music_Store_Query.sql` | All SQL queries — CTEs, Window Functions, Subqueries |
| `Music_Store_database.sql` | Complete database schema & data |
| `music store data.zip` | Raw dataset files |
